# django-light
The bare minimum required to run a Django 1.8 project without errors. (Hello World for Django)

# Running it
Use a virtualenv with Django 1.7 or higher
```python
python hello.py runserver
```

Navigate with your web browser to
```
http://localhost:8000/
```
